Tech Stack:

    Frontend:
        Next.js + TypeScipt
        Tailwind CSS
        Shadcn UI
        React Icons

    Backend:
        Next.js API Routes + TypeScipt
    
    Database:
        PostgreSQL
        Prisma ORM

    State Management:
        Zustand

    Authentication:
        NextAuth.js

Installation:

    1. npx create-next-app@latest
    2. npm install @prisma/client prisma next-auth @next-auth/prisma-adapter zustand shadcn/ui react-icons
